# COMP110 Course Survey Data Analysis

## Overview
Analyzed survey data from COMP110 students to identify trends in course satisfaction, recommendation rates, and demographics.

## Tools
- Python
- Pandas
- Matplotlib

## Analysis
- Cleaned survey data
- Calculated major distributions
- Computed average recommendation and interest ratings
- Identified trends across majors

## Results
- Neuroscience, Biology, and Economics were the most represented majors.
- Identified differences in recommendation and interest ratings that informed suggestions for course improvement.
